# TLR2 Position Embedding Analysis

This project is designed to analyze TLR2 position embeddings.

## Prerequisites

- **Python 3.11** is required to run this project.

## Installation

Install the required Python packages using `pip`:

```bash
pip install -r requirements.txt
```

## Visualization

To view the comparison of results under different window sizes, please visit the [GitHub Page](https://ucas-2.github.io/tlr2_pos_embed_ana/).
